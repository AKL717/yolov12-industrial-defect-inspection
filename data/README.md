# Dataset

This project uses the public NEU-DET steel surface defect dataset.

Dataset reference:

- https://github.com/Charmve/Surface-Defect-Detection/tree/master/NEU-DET

Expected local layout after downloading/converting annotations:

```text
data/datasets/NEU-DET/
├── train/
│   ├── images/
│   └── labels/
├── valid/
│   ├── images/
│   └── labels/
├── ANNOTATIONS/
└── data.yaml
```

The dataset itself is not committed to keep the repository lightweight.
